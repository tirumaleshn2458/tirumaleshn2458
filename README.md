## What I'm about

I'm a full-stack engineer who likes the hard parts: real-time systems, backend architecture, and the performance and reliability work that only shows up once real users hit your product.

As a **Founding Software Engineer at RivalArc**, I took a mobile social product from zero to a live App Store release, owning the full stack across React Native, FastAPI, MongoDB, Redis, and AWS. Today I work on applied-AI document pipelines, building model routing, hallucination controls, and human-in-the-loop review into production systems.

I care less about the title and more about shipping things that hold up under real load.
